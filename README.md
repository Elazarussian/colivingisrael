## Deploying to Firebase Hosting

Prerequisites:

- Install the Firebase CLI: `npm install -g firebase-tools` and authenticate with `firebase login`.
- Ensure you have access to the Firebase project `israelcoliving` or change the project id in `.firebaserc`.

To build and deploy the app to Firebase Hosting run:

```powershell
npm run deploy
```

This runs a production build into `dist/coliving-israel` and deploys the `hosting` target defined in `firebase.json`.
